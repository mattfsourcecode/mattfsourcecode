## Programmed 2013 - 2024

_[Play With Sound and the Zero Gravity Filter](https://playwithsound.live)_ is a freeware portfolio web app that leverages the [Service Worker API](https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API) and applies simulated gravity and velocity properties to [audio filters](<https://en.wikipedia.org/wiki/Filter_(signal_processing)>) through user interactions while displaying a configurable ultra [hi-res](https://en.wikipedia.org/wiki/High-resolution_high-definition)
responsive [Fast Fourier transform](https://en.wikipedia.org/wiki/Fast_Fourier_transform) algorithm (a spectrum) developed optimally in the [JavaScript](https://javascript.info/) programming language. This innovative method creates a vast array of hidden approaches for generating sound outputs from within the application. Audio effects can even be produced from web browser [`resize`](https://developer.mozilla.org/en-US/docs/Web/API/Window/resize_event) events! Activation of device low power mode throttles the application's performance.
